    Experiência

    Estágio em Desenvolvimento de Software

    Laboratório de Sistemas Inteligentes e Robótica - SIRLab, Petrópolis, RJ
    (Abril de 2014 - Dezembro de 2015)

    Trabalhou no SIRLab como desenvolvedor de software em linguagem C++. O projeto consistia em desenvolver um sistema autônomo de futebol de robôs para a categoria IEEE Very Small Size Soccer. Dentre as áreas que constituiam o desenvolvimento do sistema SIRSoccer, ficou encarregado da parte de sistemas e controle.

    O trabalho teve início com o estudo de tópicos de geometria analítica para a criação de um namespace com funções que calculariam distâncias e angulações. Após o funcionamento de uma estratégia base capaz de fazer estimações próximas ao modelo real, deu-se início a pesquisa do modelo numérico do sistema dos robôs e de controladores capazes de contornar as mudanças abruptas em seus pontos de operação. Inicialmente, os problemas decorrentes das falhas no sistema eram tratados com um controle clássico, linear e univariável, o PID.

    Com o tempo, percebeu-se que o uso do PID apenas seria efetivo de forma embarcada para fazer ajustes na velocidade angular das rodas, ao mesmo tempo que seria necessário um controle não linear na correção das coordenadas cartesianas e da angulação.
